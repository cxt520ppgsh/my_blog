### 前后端分别部署
* go run后端 ，跑在4000端口

* cd front_end, npm run dev ， 前端跑在8000端口

* 访问localhost:8000，通过js访问后端


### 后端部署前端

* cd front_end, npm build dev ， 将dist下的index.html拷贝到root/view下，将dist下的static合并到/root/static下，通过router定位到该html

* go run后端 ，跑在4000端口，通过后端访问到入口html
